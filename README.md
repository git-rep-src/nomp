# NOMP - Ncurses OpenVAS Manager Protocol

A simple Ncurses OpenVAS client

## INSTALLATION

```shell
ncurses  >= 6.0
openssl  >= 1.0.2
libxml++ >= 3.0
```
```shell
make
make install
```
## CONFIGURATION

~/.nomp/config

```shell
host=host
port=port
username=username
password=password
```
